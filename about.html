flatlakeToRSS2 0.0.1

Working oroof of concept.

Authors

Maintainers

  • R. S. Doiel Doiel, https://orcid.org/0000-0003-0900-6903

This is a demo of using the FlatLake generated JSON API to produce an RSS 2.0 feed. It's a working proof of concept only.

The Deno+TypeScript program that will convert a FlatLake generated `api/all/page-1.json` file into RSS 2.0 XML.
